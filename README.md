## ShopBot AI Customer Support Chatbot

### 📌 About

ShopBot is an AI-powered customer support chatbot for e-commerce websites. It answers common customer questions such as:

* 📦 Order Tracking
* 🚚 Shipping Information
* 🔄 Return Policy
* 💰 Refunds
* 💳 Payment Methods
* 🎧 Customer Support

The chatbot uses **Sentence Transformers** and **Cosine Similarity** to find the best answer from a predefined FAQ dataset. 

---

## 🛠 Requirements

Install Python packages:

```bash
pip install gradio pandas numpy sentence-transformers scikit-learn
```

---

## ▶️ Run the Program

1. Save the file as `shopbot.py`
2. Open Terminal / CMD
3. Run:

```bash
python shopbot.py
```

4. Open the local URL shown in the terminal (usually `http://127.0.0.1:7860`).
